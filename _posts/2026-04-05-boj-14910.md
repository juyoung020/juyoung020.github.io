## 🔗 문제 링크
[BOJ 14910 - 오르막](https://www.acmicpc.net/problem/14910)

## 📝 문제 설명
주어진 N개의 정수가 비내림차순으로 나열되어 있는지 판정하는 프로그램을 작성하시오. N개의 수 A1, A2, ..., AN이 A1 ≤ A2 ≤ ... ≤ AN 을 만족하는 것을 비내림차순이라고 한다.

## ✅ 풀이


## 💻 코드
```python
import sys
data = list(map(int, sys.stdin.readline().split()))

if len(data) == 1:
    print("Good")
else:
    boolean = True
    for i in range(len(data) - 1):
        if data[i] > data[i+1]:
            boolean = False
            break
            
    if boolean:
        print("Good")
    else:
        print("Bad")
```
