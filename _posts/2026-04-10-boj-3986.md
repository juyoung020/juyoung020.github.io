## 🔗 문제 링크
[BOJ 3986 - 좋은 단어](https://www.acmicpc.net/problem/3986)

## 📝 문제 설명
📝 문제 설명
단어 위로 곡선을 그어 같은 글자끼리 짝을 지을 때, 선이 교차하지 않으면서 모든 글자가 짝이 맞는지 확인하여 좋은 단어의 개수를 세는 프로그램

## 💡 접근
선이 교차하지 않으려면 가장 최근에 나온 글자가 바로 다음에 다시 나오거나, 안쪽 짝들이 먼저 다 사라져야 함. 스택의 괄호 쌓기로 풀 수 있음

## ✅ 풀이
1. 글자를 하나씩 확인하면서 스택의 맨 위 글자와 지금 글자가 같은지 비교한다.
2. 만약 글자가 같다면 짝이 맞으므로 스택에서 해당 글자를 꺼내서 없앤다.
3. 글자가 다르거나 스택이 비어있으면 나중에 짝을 맞추기 위해 스택에 넣는다.
4. 모든 글자를 다 돌았을 때 스택이 텅 비어있다면, 짝이 맞은 것이므로 개수를 하나 늘려준다.

## 💻 코드
```python
import sys

n = int(sys.stdin.readline())
cnt = 0

for _ in range(n):
    word = sys.stdin.readline().strip()
    stk = []
    
    for char in word:
        if stk and stk[-1] == char:
            stk.pop()
        else:
            stk.append(char)
            
    if not stk:
        cnt += 1

print(cnt)
```
