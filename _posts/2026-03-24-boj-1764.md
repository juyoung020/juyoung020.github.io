## 🔗 문제 링크
[BOJ 1764 - 듣보잡](https://www.acmicpc.net/problem/1764)

## 📝 문제 설명
김진영이 듣도 못한 사람의 명단과, 보도 못한 사람의 명단이 주어질 때, 듣도 보도 못한 사람의 명단을 구하는 프로그램을 작성하시오.

## 💻 코드
```python
n, m = map(int, input().split())


n_set = set()
for _ in range(n):
    n_set.add(input())


answer = []
count = 0
for _ in range(m):
    m_data = input()
    if m_data in n_set:
        count +=1
        answer.append(m_data)

answer.sort()
print(count)
print("\n".join(answer))
```
