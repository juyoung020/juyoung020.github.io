## 🔗 문제 링크
[BOJ 14425 - 문자열 집합](https://www.acmicpc.net/problem/14425)

## 📝 문제 설명
총 N개의 문자열로 이루어진 집합 S가 주어진다.

입력으로 주어지는 M개의 문자열 중에서 집합 S에 포함되어 있는 것이 총 몇 개인지 구하는 프로그램을 작성하시오.

## 💻 코드
```python
input = __import__('sys').stdin.readline

n, m = map(int, input().split())

s = set()

for _ in range(n):
    s.add(input().strip())

answer = 0
for _ in range(m):
    check = input().strip()
    if check in s:
        answer +=1

print(answer)
```
