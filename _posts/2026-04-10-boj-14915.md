## 🔗 문제 링크
[BOJ 14915 - 진수 변환기](https://www.acmicpc.net/problem/14915)

## 📝 문제 설명
정수 m, n을 입력 받아, 10진수 m을 n진수로 바꾸어 출력하는 프로그램을 작성하시오.

## 💡 접근


## ✅ 풀이


## 💻 코드
```python
import sys

input_data = sys.stdin.readline().split()

m = int(input_data[0])
n = int(input_data[1])

if m == 0:
    print(0)
else:
    digits = "0123456789ABCDEF"
    result = ""

    while m > 0:
        result += digits[m % n]
        m //= n

    print(result[::-1])
```
