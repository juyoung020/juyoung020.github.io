## 🔗 문제 링크
[BOJ 1264 - 모음의 개수](https://www.acmicpc.net/problem/1264)

## 📝 문제 설명
영문 문장을 입력받아 모음의 개수를 세는 프로그램을 작성하시오. 모음은 'a', 'e', 'i', 'o', 'u'이며 대문자 또는 소문자이다.

## 💻 코드
```python
input = __import__('sys').stdin.readline
data = ['a', 'e', 'i', 'o', 'u']

while True:
    n = input().rstrip()

    if n == '#':
        break

    count = 0
    lower_n = n.lower()
    
    for char in lower_n:
        if char in data:
            count += 1

    print(count)
```
