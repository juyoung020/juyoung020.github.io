## 🔗 문제 링크
[BOJ 1371 - 가장 많은 글자](https://www.acmicpc.net/problem/1371)

## 📝 문제 설명
영어에서는 어떤 글자가 다른 글자보다 많이 쓰인다. 예를 들어, 긴 글에서 약 12.31% 글자는 e이다.

어떤 글이 주어졌을 때, 가장 많이 나온 글자를 출력하는 프로그램을 작성하시오.

## 💻 코드
```python
input = __import__('sys').stdin.read

data = input()

counts = [0] * 26

for char in data:
    if char.islower():
        counts[ord(char) - ord('a')] += 1

max_val = max(counts)

answer = ""
for i in range(26):
    if counts[i] == max_val:
        answer += chr(i + ord('a'))
print(answer)
```
