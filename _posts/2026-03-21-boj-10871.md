## 🔗 문제 링크
[BOJ 10871 - X보다 작은 수](https://www.acmicpc.net/problem/10871)

## 📝 문제 설명
정수 N개로 이루어진 수열 A와 정수 X가 주어진다. 이때, A에서 X보다 작은 수를 모두 출력하는 프로그램을 작성하시오.

## 💡 접근


## ✅ 풀이


## 💻 코드
```python
N, X = map(int, input().split())
A = list(map(int, input().split()))

min_num = [str(i) for i in A if i < X]

print(' '.join(min_num))
```
