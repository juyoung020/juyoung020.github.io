## 🔗 문제 링크
[BOJ 5218 - 알파벳 거리](https://www.acmicpc.net/problem/5218)

## 📝 문제 설명
길이가 같은 두 단어가 주어졌을 때, 각 단어에 포함된 모든 글자의 알파벳 거리를 구하는 프로그램을 작성하시오.

두 글자 x와 y 사이의 알파벳 거리를 구하려면, 먼저 각 알파벳에 숫자를 할당해야 한다. 'A'=1, 'B' = 2, ..., 'Z' = 26. 그 다음 y ≥ x인 경우에는 y-x, y < x인 경우에는 (y+26) - x가 알파벳 거리가 된다.

예를 들어, 'B'와 'D' 사이의 거리는 4 - 2 = 2이고, 'D'와 'B' 사이의 거리는 (2+26) - 4 = 24이다.

## 💻 코드
```python
n = int(input())

temp = 0
for _ in range(n):
    alpha = input().split()
    word1 = list(alpha[0])
    word2 = list(alpha[1])
    answer = []

    for i in range(len(word1)):
        x = ord(word1[i]) - 64
        y = ord(word2[i]) - 64

        if  y >= x:
            temp = y - x
        else:
            temp = (y+26) - x

        answer.append(temp)
    print("Distances:", *answer)
```
