## 🔗 문제 링크
[BOJ 9498 - 시험 성적](https://www.acmicpc.net/problem/9498)

## 📝 문제 설명
시험 점수를 입력받아 90 ~ 100점은 A, 80 ~ 89점은 B, 70 ~ 79점은 C, 60 ~ 69점은 D, 나머지 점수는 F를 출력하는 프로그램을 작성하시오.

## ✅ 풀이


## 💻 코드
```python
score = int(input())

if score >= 90:
    print("A")

elif score >= 80:
    print("B")

elif score >= 70:
    print("C")

elif score >= 60:
    print("D")

else:
    print("F")
```
