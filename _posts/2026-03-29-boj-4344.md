## 🔗 문제 링크
[BOJ 4344 - 평균은 넘겠지](https://www.acmicpc.net/problem/4344)

## 📝 문제 설명
대학생 새내기들의 90%는 자신이 반에서 평균은 넘는다고 생각한다. 당신은 그들에게 슬픈 진실을 알려줘야 한다.

## 💻 코드
```python
C = int(input())

for _ in range(C):
    data = list(map(int, input().split()))
    N = data[0]
    scores = data[1:]

    av = sum(scores)/N

    count = 0

    for s in scores:
        if s > av:
            count +=1

    rat = count /N *100
    print(f"{rat:.3f}%")
```
