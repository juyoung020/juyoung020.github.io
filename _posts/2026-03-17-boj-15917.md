## 🔗 문제 링크
[BOJ 15917 - 노솔브 방지문제야!!](https://www.acmicpc.net/problem/15917)

## 📝 문제 설명
여러분은 Q개의 쿼리를 수행해야 합니다. 수행해야 하는 쿼리는 다음과 같습니다.

어떤 수 a를 2의 거듭제곱 꼴로 나타낼 수 있는가?

## ✅ 풀이


## 💻 코드
```python
import sys

QandDATA = sys.stdin.read().split()

if QandDATA:
    count = int(QandDATA[0])
    answer = []

    for i in range(1, count + 1):
        a = int(QandDATA[i])

        if a & (-a) == a:
            answer.append("1")
        else:
            answer.append("0")

sys.stdout.write("\n".join(answer))
```
