## 🔗 문제 링크
[BOJ 1330 - 두 수 비교하기](https://www.acmicpc.net/problem/1330)

## 📝 문제 설명
두 정수 A와 B가 주어졌을 때, A와 B를 비교하는 프로그램을 작성하시오.

## 💡 접근


## ✅ 풀이


## 💻 코드
```python
A, B = map(int, input().split())

if A > B:
    print(">")
elif A < B:
    print("<")

else:
    print("==")
```
