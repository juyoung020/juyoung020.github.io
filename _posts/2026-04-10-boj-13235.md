## 🔗 문제 링크
[BOJ 13235 - 팰린드롬](https://www.acmicpc.net/problem/13235)

## 📝 문제 설명
팰린드롬은 앞에서부터 읽을 때와 뒤에서부터 읽을 때가 똑같은 단어를 의미한다. 예를 들어, eve, eevee는 팰린드롬이고, eeve는 팰린드롬이 아니다. 단어가 주어졌을 때, 팰린드롬인지 아닌지 판단해보자.

## ✅ 풀이


## 💻 코드
```python
import sys

word = sys.stdin.readline().strip()

if word == word[::-1]:
    print("true")
else:
    print("false")
```
