## 🔗 문제 링크
[BOJ 1813 - 논리학 교수](https://www.acmicpc.net/problem/1813)

## 📝 문제 설명
논리학 교수 양항승은 칠판에 다음과 같은 내용을 썼다.

정확하게 a개의 말은 참이다.
정확하게 b개의 말은 참이다.
정확하게 c개의 말은 참이다.
...
...
...

a, b, c는 정수이다. 그리고 나서 항승이는 칠판에 작성한 내용 중에 총 몇 개가 참인지 알아내는 사람은 A+을 받는다. 입력으로 항승이가 작성한 내용에 있는 정수가 주어진다. 예를 들면, "정확하게 i개의 말은 참이다" 에서 i가 입력으로 주어진다. 

항승이가 칠판에 작성한 내용이 주어졌을 때, 총 몇 개의 내용이 참인지 구해보자.

## 💻 코드
```python
input = __import__('sys').stdin.readline

n = int(input())
numbers = list(map(int, input().split()))

answer = -1

for k in range(n + 1):
    count = 0
    for num in numbers:
        if num == k:
            count += 1
            
    if count == k:
        answer = max(answer, k)

print(answer)
```
