## 🔗 문제 링크
[[BOJ 14579] N 찍기](https://www.acmicpc.net/problem/14579)

## 📝 문제 설명
a, b (1 ≤ a < b ≤ 1000)가 주어졌을 때

(1+2+…+a) * (1+2+…+(a+1)) * … * (1+2+…+(b-1)) * (1+2+…+b)의 값을 계산하는 문제


## 💡 접근
1~a까지 1~b까지를 다 더한 값을 곱하는거
a < b

등차수열의 합
(i * (i+1)) // 2

a~b까지 각각의 등차수열의 합을 곱하면 되겠다

## ✅ 풀이
공백을 기준으로 2입력 받기
n = map(int, input().split())
반복문으로 a~b까지의 각각의 숫자인 등차수열의 합을 곱하기
14579로 나눈 나머지값으로 값 줄이기

## 💻 코드
```python
a, b = map(int, input().split())
answer = 1

for i in range(a, b+1):
    sum_temp = i*(i+1)//2
    answer = (answer * sum_temp)%14579

print(answer)
```
