## 🔗 문제 링크
[BOJ 2869 - 달팽이는 올라가고 싶다](https://www.acmicpc.net/problem/2869)

## 📝 문제 설명
달팽이가 낮에는 A만큼 올라가고 밤에는 B만큼 미끄러질 때, 목표 높이 V에 도달하는 최소 일수를 구하는 프로그램

## 💡 접근
반복문을 쓰면 시간 초과가 나니까 수식으로 한 번에 계산,
마지막 날 정상에 찍는 순간은 미끄러지지 않음

## ✅ 풀이
1. 전체 높이 V에서 마지막 날 A를 뺀 값을 하루 실질 이동 거리(A-B)로 나눠야겠다.
2. 나누어떨어지지 않으면 하루가 더 필요하니까 올림 처리를 해야겠다.
3. 계산된 값에 마지막 날 하루(+1)를 더해서 출력해야겠다.

## 💻 코드
```python
import math
input = __import__('sys').stdin.readline

A, B, V = map(int, input().split())
rast_day = math.ceil((V-A)/(A-B))

print(rast_day + 1)
```
