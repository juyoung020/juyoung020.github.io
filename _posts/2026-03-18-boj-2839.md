## 🔗 문제 링크
[BOJ 2839 - 설탕 배달](https://www.acmicpc.net/problem/2839)

## 📝 문제 설명
입력을 받아서 5나 3을 섞어서 나눠 떨어지면 그 몫을 출력하고
나머지가 있으면 -1를 출력하는 문제

## 💡 접근
입력 N을받아서
1순위는 5로 나눠 떨어지면 카운트하고
2순위는 3으로 나눠 떨어지면 카운트하게 하고
3순위는 5랑 3을 적절히 섞어도 나머지가 나오면 N을 -1로 바꿔야겠다

## ✅ 풀이
1. while문 사용 N이 0보다 클때 반복을 해야겠다.
2. 만약 5로 나눠떨어지면 -5를 빼고 count를 1올리는걸 안나눠질때까지 반복해야겠다
3. 3으로 나눠떨어지면 -3을 빼고 count를 1 올려야겠다
4. n이 음수가 되면 -1을 출력해야겠다.

## 💻 코드
```python
N = int(input())

count = 0
while N > 0:
    if N % 5 == 0:
        N -= 5
        count += 1


    else: 
        N -= 3
        count +=1

if N < 0:
    count = -1
    print(count)

else:
    print(count)
```
